# 🤖 AI HR Assistant — Análise de Currículos com IA (RAG + LLM Local)

Este projeto apresenta uma aplicação de **Inteligência Artificial Generativa** para análise de currículos utilizando **RAG (Retrieval-Augmented Generation)** executando **100% localmente**.

A aplicação funciona como um **Assistente de Recursos Humanos baseado em IA**, permitindo analisar currículos e responder perguntas em linguagem natural sobre candidatos.

O objetivo é **auxiliar recrutadores na triagem inicial de currículos**, tornando o processo seletivo mais rápido e eficiente.

---

# 🚀 Tecnologias Utilizadas

- Python  
- Streamlit  
- Ollama  
- Llama 3  
- LlamaIndex  
- HuggingFace Embeddings  
- RAG (Retrieval-Augmented Generation)
